# STARTOMB EXPLORER
A text-based RPG created for weeks 11/12 at Specialisterne Academy.

Created by SpecAcad-Jakob, Thor271998, Ewrefads, CarolineKjaersgaard, & AsgerMoellerSpec.

# To Run:
Make sure C#, .NET, MySql, html and css are installed on your system.
Open a terminal and navigate to the Backend directory.
Enter "dotnet watch run" to start the backend and API.
Open a terminal and navigate to the Frontend directory.
Enter "dotnet run" to start the frontend.

# Description:
System should be able to:
Read predefined adventure elements (rooms, enemies, etc.) from csv-files.
Populate a database with this data.
Run the game logic and draws from the aforementioned database.
Allow a user to interact with the game through an API.
Open a browser-based GUI through which to experience the game.
Let you have a good time dying in space.
